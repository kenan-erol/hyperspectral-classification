Methodology Overview

The project workflow involves several key steps:

  1. Data Conversion (.raw/.hdr to .npy)
  2. Pill Segmentation using SAM 2
  3. Patch Extraction
  4. Data Augmentation (for Real/Fake task)
  5. Label Generation
  6. CNN Model Training (Drug Type and Real/Fake)
  7. Evaluation
